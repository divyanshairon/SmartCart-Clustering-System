# 🛒 SmartCart Customer Segmentation (Clustering System) using Unsupervised Learning

## 📌 Project Overview
This project focuses on customer segmentation using Unsupervised machine learning techniques. The goal is to group customers based on purchasing behavior to help businesses improve marketing strategies. 2 Unsupervised Machine Learning models have been used: K-Means Clustering and Hierarchial Clustering. 

## 🚀 Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Clustering using K-Means and Hierarchial
- Visualization of customer segments

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
